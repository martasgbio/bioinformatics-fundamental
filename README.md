## ğŸ›   Pipelines Included

### 1. Basic FASTA Sequence Analysis

**Location:** `pipelines/basic_analysis/`

This pipeline performs an exploratory analysis of DNA sequences in FASTA format, including:
- FASTA statistics (length, GC content)
- GC content analysis using a sliding window
- ORF (Open Reading Frame) detection

This pipeline demonstrates:
- Sequence parsing
- Basic genomic feature analysis
- Modular pipeline design

---

### 2. Variant Analysis

**Location:** `pipelines/variant_analysis/`

This pipeline performs a basic analysis of genetic variants from a VCF file, including:
- Parsing variant information
- Basic variant statistics
- Summary of SNPs and indels

This pipeline serves as an introduction to variant processing workflows and provides
a foundation for future extensions such as cancer genomics.
